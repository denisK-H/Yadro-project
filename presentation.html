Автоматизация измерения АЧХ усилительного каскада

Программный комплекс для управления генератором, осциллографом, съема точек АЧХ и поиска частоты единичного усиления.

Цель работы

  • Автоматизировать лабораторное измерение амплитудно-частотной характеристики.
  • Исключить ручной перебор частот и ручную запись значений.
  • Получать воспроизводимые данные в формате CSV.
  • Автоматически находить область, где коэффициент усиления становится равным единице.
  • Строить графики по результатам измерений.

Используемые приборы

Генератор сигналов

Формирует синусоидальный входной сигнал. Частота изменяется программно через SCPI-команды.

Осциллограф OWON XDS3064E

Измеряет входной и выходной размах напряжения.

  • CH1: Vin
  • CH2: Vout
  • Режим каналов: AC, смещение 0

Роль Go-программы

  • Читает параметры эксперимента из config.yaml.
  • Подключается к REST API моста.
  • Отправляет команды настройки приборов.
  • Управляет частотным планом эксперимента.
  • Запрашивает Vin и Vout.
  • Рассчитывает Ku = Vout / Vin и Ku в дБ.
  • Сохраняет результат в results.csv.

Роль Python REST API

generator.py работает как локальный мост между Go-программой и измерительными приборами.

  • /connect - подключение к прибору по VISA resource.
  • /write - отправка SCPI-команды.
  • /query - команда с текстовым ответом.
  • /query_number - чтение числового результата измерения.
  • /close_all - закрытие соединений.

Этапы работы программы

  1. Чтение config.yaml.
  2. Подключение генератора и осциллографа.
  3. Настройка осциллографа: CH1/CH2, AC, offset 0.
  4. Настройка генератора: синус, амплитуда, выход ON.
  5. Выбор очередной частоты.
  6. Подстройка временного и вертикального масштаба.
  7. Измерение Vin и Vout.
  8. Расчет Ku и запись точки в CSV.

Методика измерений

На каждой частоте генератор устанавливает синусоидальный сигнал, а осциллограф измеряет размах напряжения на входе и выходе.

Vin  - входное напряжение, Vpp
Vout - выходное напряжение, Vpp
Ku   = Vout / Vin
Ku_dB = 20 * log10(Ku)

Для устойчивости используются задержки установления сигнала и повторные попытки измерения.

Масштабирование осциллографа

  • Штатный autoscale отключен, потому что он может обрезать сигнал.
  • Временная развертка выбирается так, чтобы на экране помещалось несколько периодов.
  • Вертикальный масштаб CH1 и CH2 подбирается по измеренному Vpp.
  • Если значение некорректно или сигнал не помещается, масштаб увеличивается.
  • Смещение каналов выставляется в ноль.

Поиск частоты единичного усиления

Частота единичного усиления соответствует условию Ku = 1, то есть Ku_dB = 0 dB.

  • Поиск выполняется по логарифмической сетке частот.
  • Если переход найден между двумя точками, программа уточняет область дополнительными измерениями.
  • Диапазон и точность задаются в секции unity_search.

Формат результата

Основной файл результата: results.csv.

  • frequency_hz - частота.
  • vin_vpp, vout_vpp - входное и выходное напряжение.
  • gain, gain_db - коэффициент усиления.
  • ok - успешность измерения.
  • point_type - тип точки.
  • vin_raw, vout_raw - исходные ответы осциллографа.

Пример работы

В одном из запусков было получено 48 измеренных точек.

f = 1000 Hz
Vin = 0.260 Vpp

Ku = 4.31
Ku_dB = 12.68 dB

Область единичного усиления:

4.6639 MHz: Ku = 1.038
4.8919 MHz: Ku = 0.943

Частота Ku = 1 находится примерно между 4.66 и 4.89 МГц.
