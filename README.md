# Автоматическое исследование каскада с общим эмиттером

Проект измеряет АЧХ каскада с общим эмиттером с помощью генератора и осциллографа.

## Кто что делает

### `bridge.py`
Python-мост к приборам.

Что делает:
- подключает генератор и осциллограф через `pyvisa`
- отправляет SCPI-команды
- читает ответы приборов
- отдаёт данные в Go по HTTP

### `main.go`
Основная логика измерения.

Что делает:
- читает `config.yaml`
- подключается к `bridge.py`
- настраивает генератор
- делает sweep по частоте
- измеряет `Vin` и `Vout`
- считает коэффициент усиления `Ku = Vout / Vin`
- сохраняет результат в `results.csv`

### `plot_ach.py`
Построение графиков из `results.csv`.

Что делает:
- читает CSV
- строит график АЧХ
- строит график в дБ
- строит график `Vin` и `Vout`

### `config.yaml`
Конфиг эксперимента.

Что хранит:
- адрес bridge
- resource строку приборов
- SCPI-команды
- диапазон частот
- число точек
- задержки и параметры измерения

## Как запускать

### 1. Установить зависимости Python

```bash
pip install pyvisa pyvisa-py flask matplotlib
```
2. Установить зависимости Go

go mod tidy
3. Запустить Python bridge
python bridge.py
4. Запустить измерение
go run .\main.go

После этого появится файл:

results.csv
5. Построить графики
python plot_ach.py

После этого появятся файлы:

ach_linear.png
ach_db.png
vin_vout.png
Порядок работы
bridge.py поднимает HTTP-сервис для работы с приборами
main.go через него управляет генератором и осциллографом
main.go сохраняет измерения в results.csv
plot_ach.py строит графики по CSV
